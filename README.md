- **About** — Bio and background

- **Short Form Content** — Social media reels and edits
- **Student Projects** — *Have a Good Trip* student film BTS
- **Contact** — Email, phone, socials

## Adding Photos

Drop images into an `assets/` folder and replace the `<div class="ph">` placeholders in `index.html` with:

```html
<img src="assets/concerts/photo.jpg" alt="Description">
```
